Alinhamento de textos usando CSS

Este primeiro páragrafo terá o atendimento padrão, à esquerda.

O segundo parâmetro aguardar a direita.

O terceiro parágrafo terá acompanhamento centrado.

O último parágrafo terá o alinhamento justificado. Para que você perceba isso, terei que colocar um texto um pouco maior. Caso você não perceba que o controlado está sendo feito nas duas laterais, tente diminuir um pouco o tamanho do seu navegador. Esse tipo de segurança é bastante utilizado para textos longos e grandes parágrafos, para dar uma sensação de simetria.
